📚 AVAILABLE FUNCTIONS
═══════════════════════════════════════════════════════════════════════════════

JavaScript Global Functions:

1. toggleSidebar()
   └─ Toggle sidebar visibility (desktop: toggle, mobile: slide)

2. toggleUserDropdown()
   └─ Toggle user profile dropdown menu

3. toggleSubMenu(event, menuId, chevronId)
   └─ Toggle submenu expansion with animation

4. loadPage(url, title = '')
   └─ Load page via AJAX with smooth transition
   └─ Update browser title
   └─ Show/hide loader

5. submitFormAjax(formId, successCallback)
   └─ Submit form data via AJAX
   └─ Process multipart data
   └─ Execute callback on success

6. reinitializeBootstrap()
   └─ Reinit tooltips and popovers

═══════════════════════════════════════════════════════════════════════════════

Tablet:  640-1024  (md:)   - Two columns, sidebar visible
Desktop: > 1024px  (lg:)   - Multi-column, full width

Usage in Tailwind:

  class="hidden md:block"
